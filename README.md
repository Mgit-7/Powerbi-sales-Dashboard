#  Power BI Dashboard Project (PL-300)

##  Overview

This project was created as part of Microsoft PL-300 Power BI Data Analyst training.
It demonstrates building an interactive dashboard from a report using Power BI Service.

## 🛠️ Tools Used

* Power BI Desktop
* Power BI Service

## Features

* Created report visuals (charts, KPIs)
* Pinned visuals to dashboard
* Used Q&A to generate insights
* Designed dashboard layout for business users
